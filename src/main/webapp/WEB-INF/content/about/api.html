API

  • Работа с пользователями
    • /api/user/{id}.json (GET) - Получить пользователя по id
    • /api/user/{id}.json (PUT) - Изменить пользователя с указанным id
    • /api/user.json (POST) - Добавить пользователя с указанным id
    • /api/user/{id}/password/{password}.json (POST) - Изменить пароль у пользователя с указанным id
    • /api/admin/blocker/{id}.json (PUT) - Заблокировать пользователя с указанным id
  • Работа с каталогами
    • /api/catalog/{name}/list.json (GET) - Получить указанный каталог
    • /api/catalog/{name}/{id}.json (GET) - Получить элемент с указанным id из каталога
  • Работа с новостями
    • /api/news/list.json (GET) - Получить список новостей
    • /api/admin/news.json (POST) - Добавить новость
    • /api/admin/news.json (PUT) - Изменить новость
  • Работа с задачами
    • /api/user/{user}/task/{id}.json (GET) - Получить задачу указанного пользователя по id
    • /api/user/{user}/task/list.json (GET) - Получить список задач указанного пользователя
    • /api/user/{user}/task.json (POST) - Добавить задачу пользователю
    • /api/user/{user}/task/{id}.json (PUT) - Изменить задачу
    • /api/user/{user}/task/{id}.json (DELETE) - Удалить задачу (Пометить как удаленную)
