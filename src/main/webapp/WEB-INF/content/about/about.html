О проекте

Добро пожаловать на "Not Today" - веб-сервис позволяющий управлять списками дел

Перед Вами мой тестовый проект, на котором я собираюсь испытывать новые, интересные мне технологии и оттачивать свой навых программирования
Первое время сервис будет работать в тестовом режиме (возможно это навсегда :), а значит возможны поломки и много-много багов :)
Я буду очень признателен Вам за любую критику, а так же за помощь в развитии проекта. Ваши предложения можно присылать на почту: vlasovi.ilya@gmail.com, а сам можно скачать на github.
Архитектура сервиса устроенна следующим образом:
  • Клиент - это приложение на Angular, обзщающееся с сервером по протоколу REST
  • Сервер - это приложение на Java + Spring MVC
Подробнее про api сервера можно узнать по ссылке: API
